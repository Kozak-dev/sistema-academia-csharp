# 🏋️ Sistema de Gerenciamento de Academia

Sistema desenvolvido em **C# (.NET)** com foco no aprendizado de **Programação Orientada a Objetos (POO)**.

O projeto simula o funcionamento de uma academia, permitindo o gerenciamento de alunos, personal trainers, planos, matrículas, pagamentos, presenças, avaliações físicas e relatórios.

---

## 📋 Funcionalidades

- Cadastro de alunos
- Cadastro de personal trainers
- Cadastro de planos
- Matrículas
- Pagamentos
- Controle de presença
- Avaliações físicas
- Cadastro de exercícios
- Cadastro de treinos
- Relatórios administrativos

---

## 🛠 Tecnologias Utilizadas

- C#
- .NET
- Programação Orientada a Objetos
- Console Application

---

## 📚 Recursos Aplicados

- Encapsulamento
- Herança
- Composição entre objetos
- Construtores
- Collections (`List<T>`)
- CRUD completo
- Separação em Models, Services e Helpers
- Regras de negócio

---

## 🎯 Objetivo

Este projeto foi desenvolvido para consolidar os conhecimentos em C# e Programação Orientada a Objetos antes do desenvolvimento de uma versão Web utilizando **ASP.NET Core Web API**, **Entity Framework Core** e **SQL Server**.
